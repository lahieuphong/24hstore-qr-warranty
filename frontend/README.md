# 24hStore QR Warranty - Frontend cũ

Đây là mã giao diện tách riêng trước đây, được giữ lại để tham khảo. Trang tra cứu công khai hiện đã tích hợp vào `backend/` và chạy tại:

- Tra cứu IMEI: `http://127.0.0.1:8000/check`
- Tra cứu bằng QR: `http://127.0.0.1:8000/check/{qr_token}`

Không cần cài dependency hoặc chạy ứng dụng trong thư mục này cho luồng local hiện tại.
